## About

The `qdatamatrix` package provides a graphical widget to view and edit a `datamatrix` object.

## Dependencies

- Python 2.7 or >= 3.4
- [python-datamatrix](https://github.com/smathot/python-datamatrix/)
- [qtpy](https://pypi.python.org/pypi/QtPy/)
